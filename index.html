Aprendizado sobre git hub

git init: cria um versionamento de git na sua máquina

git status: para ver arquivos que não foram adicionados ou os commits que faltam

git log: vê os commits feitos no projetos

git add: adiciona arquivos para serem comitados

git commit: cria uma "foto" de como esta o projeto no momento

git clone: com o link do repositório é possível clona-lo para sua máquina

git push: envia os commits feito para o git hub

git pull: pega todas as alterações feitas no projeto remoto
